# Forex Simulation Engine\n\nAutomated Forex trading simulation with Technical (TA) and Fundamental (FA) analysis.\n\n## Goals\n- Simulate trades based on multi-factor signals.\n- Provide deep analytics on strategy performance.\n- Learn trajectoy of errors to improve decision making.\n\n## Architecture\n- **Data Ingestion**: Real-time news (Firecrawl) and OHLCV data.\n- **Signal Generator**: Merging TA (indicators) and FA (sentiment).\n- **Simulator**: Virtual order execution with slippage and spread modeling.\n- **Analytics**: Post-trade analysis and performance reporting.\n\n## Status\nInitial setup phase. Configured with OpenClaw MCP.
